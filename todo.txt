In frame_draw.js
Draw bits according the given bit ratio.
In case of CAN FD, the bit where the data phase starts and ends is special.
For the bit, where the data phase starts: Up to the Sample Point (SP) the bit is an Arbitration bit, from the SP the bit is a data bit.
For the bit, where the data phase ends: Up to the Sample Point (SP) the bit is an Data bit, from the SP the bit is a arbitration bit.
Show a vertiacal separtor line at the SP, with half the hight.
