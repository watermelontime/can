4. Anonymous Reach Measurement with GoatCounter

This website uses the open-source web analytics tool GoatCounter (instance: watermelon.goatcounter.com) for privacy-friendly, cookie-free reach measurement.

Types of data processed: Page visited (path/URL), referrer (if transmitted), browser/device type (abstracted from user agent), language setting, date/time. IP addresses are only used in a form that does not allow conclusions about individuals (hash/shortened derivative to count unique visits).

Purpose: Statistical, non-personal evaluation to improve structure and content and to detect technical issues.

Legal basis: Art. 6(1)(f) GDPR (legitimate interest in anonymous reach analysis without tracking profiles). No consent is required due to anonymization and the absence of cookies.

No merging: No linkage with other services or personal data.

Storage duration: Aggregated statistical data are retained permanently or until manual deletion to identify trends; any short-term raw data (e.g., for anti-abuse) are processed by GoatCounter only as long as needed for aggregation (see GoatCounter privacy notes).

Opt-out / Do Not Track: If your browser sends "Do Not Track" (DNT), the visit is not counted. More information: https://www.goatcounter.com/privacy.

Note: No cookies; if optional, consent-required features are enabled in the future, this section will be updated.
