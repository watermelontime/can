My CAN page

What can I find on this website?

This website focuses on CAN (Controller Area Network) and provides helpful tools and material.

  • CAN Register Evaluator: A tool that evaluates and finds errors in the configuration of the CAN module you use.
  • CAN XL Bit Timing Calculator: A tool that visually guides you when configuring CAN XL Bit Timing.

Who is the target of this website?

This website is built for people that are facing CAN in a real project and struggle with its configuration. Such persons are

  • Embedded software developers that directly interact with a CAN module (the piece of hardware on a microcontroller that implements the CAN communication).
  • System designers that need to decide which bit timing configuration to be used in their CAN networks
  • Anyone, who just wants to see some functional CAN bit timing configuration example

Why does this website exist?

I wanted to realize some complex project with help of AI (that I would have never started before AI was available). Further I wanted the result to be usefull for others.
